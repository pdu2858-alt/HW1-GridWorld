# 網格地圖 RL 導航器 (Gridworld RL Navigator)

這是一個基於 **Streamlit** 開發的互動式強化學習（Reinforcement Learning）教學工具，用於演示經典的網格地圖（Gridworld）問題。使用者可以手動佈置地圖，並觀察不同強化學習演算法如何計算狀態價值與尋找路徑。

## 🚀 功能特點

- **互動式地圖設定**：透過點擊網格輕鬆設定「起點」、「終點」與「障礙物」。
- **動態參數調整**：支援 5x5 至 9x9 的網格維度切換。
- **策略評估 (Policy Evaluation)**：演示 HW1-2 任務，評估隨機策略（Random Policy）下的狀態價值 $V(s)$。
- **價值迭代 (Value Iteration)**：演示 HW1-3 任務，透過迭代尋找最佳策略（Optimal Policy）與最大化價值。
- **即時視覺化**：網格直接顯示動作建議（↑ ↓ ← →）與該狀態的預期價值。

## 🛠️ 安裝與運行

### 1. 環境需求
請確保您的系統已安裝 Python 3.8+。

### 2. 安裝必要套件
```bash
pip install streamlit numpy
```

### 3. 啟動應用程式
在專案根目錄下執行：
```bash
streamlit run app.py
```

## 📖 使用指南

1. **設定環境**：
   - 使用左側側邊欄調整網格大小 (n)。
   - 依照提示依序在網格中點擊設定：**起點 (🟢)** -> **終點 (🔴)** -> **障礙物 (⬛)**。
   - 障礙物數量需達到 `n-2` 個後方可執行演算法。

2. **執行演算法**：
   - **隨機策略與評估**：點擊側邊欄按鈕，系統將以 0.25 的機率隨機選擇動作，並計算收斂後的狀態價值。
   - **價值迭代**：計算 Bellman 最佳方程式，尋找從起點到終點的最短路徑與最佳價值。

3. **查看結果**：
   - 網格中會顯示動作符號與數值。
   - 頁面下方會顯示完整的 $V(s)$ 狀態價值矩陣。

## 🧠 強化學習參數

- **折扣因子 (Gamma)**：0.9
- **回報機制 (Reward)**：
    - 到達終點：+100
    - 撞牆或障礙物：-10
    - 每移動一步：-1
